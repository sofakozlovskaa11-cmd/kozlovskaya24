Данная программа реализует TCP-сервер, работающий на локальном адресе 127.0.0.1 и порту 5000. После запуска сервер ожидает подключения клиента. Когда клиент подключается, программа принимает отправленные сообщения и выводит их на экран.

Каждое полученное сообщение сервер отправляет обратно клиенту, благодаря чему реализуется принцип работы Echo Server. Это позволяет проверить передачу данных между клиентом и сервером по протоколу TCP.

Если клиент отправляет сообщение exit, сервер завершает обработку данных, закрывает соединение и прекращает работу.

Программа написана на языке C# с использованием библиотеки .NET и классов для работы с сетевыми соединениями по протоколу TCP/IP.

Для запуска программы необходимо выполнить команду dotnet run в каталоге проекта.
